# 打地鼠小遊戲 (Whack-a-Mole)

這是一個簡單且有趣的網頁版「打地鼠」小遊戲。玩家需要在限定時間內儘可能點擊出現的地鼠以獲得高分。

## 遊戲特色

- **多種難度**：提供「簡單」、「普通」與「困難」三種難度，適合不同強度的挑戰。
- **最高分紀錄**：自動儲存您的最高得分於瀏覽器中。
- **響應式設計**：支援桌面與行動裝置遊玩。
- **即時回饋**：點擊地鼠時會有視覺與分數跳出的動畫效果。

## 難度設定

| 難度 | 地鼠停留時間 | 說明 |
| :--- | :--- | :--- |
| **簡單** | 800ms - 1500ms | 適合初學者，地鼠停留時間較長。 |
| **普通** | 450ms - 950ms | 預設難度，平衡的遊戲體驗。 |
| **困難** | 200ms - 600ms | 適合反應極快的玩家，地鼠稍縱即逝。 |

## 如何執行

您可以透過以下任一方式開啟遊戲：

1. **直接開啟**：在檔案總管中雙擊 `index.html`，使用任一現代瀏覽器（如 Chrome, Edge, Firefox）開啟即可遊玩。
2. **使用本地伺服器** (推薦)：
   - 若您安裝了 Python，可在終端機執行：`python3 -m http.server 8000`，然後開啟 `http://localhost:8000`。
   - 若您有 Node.js，可執行：`npx serve .`。

## 技術架構

- **HTML5**: 結構與語義化標籤。
- **CSS3**: 遊戲介面美化、動畫與響應式佈局。
- **JavaScript (Vanilla JS)**: 遊戲邏輯控制、計時器與難度調整。
